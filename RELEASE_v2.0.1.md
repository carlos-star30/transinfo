# Release v2.0.1

发布日期：2026-04-06

这是当前项目基于 v2.0.0 的补丁版本发布，标记为 v2.0.1。

## 本次版本包含

- 合并当前主分支已完成的前后端功能与界面调整。
- 保持 Netlify + Render 自动部署链路不变，仅通过 GitHub 推送触发构建。
- 统一前端运行时版本、页面版本角标与静态资源缓存戳。
- 补充本地临时文件忽略规则，避免发布时混入数据库缓存和本地状态文件。

## 当前发布口径

- 前端：Netlify
- 后端：Render Web Service（Docker）
- 数据库：TiDB Cloud 或其他 MySQL 兼容托管库

## 版本落点

- 前端运行时版本：`2.0.1`
- 页面版本角标：`v2.0.1`
- 静态资源缓存戳：`20260406-v2.0.1`

## 上线后建议核对

1. 登录、`/auth/me`、路径搜索是否返回 200。
2. 选定路径后，映射 / 文本 / 逻辑是否正常显示。
3. ABAP 转 SQL、程序代码导入、数据库动态查询页面是否可正常打开。
4. 前端页面右上角版本号是否显示为 `v2.0.1`。
