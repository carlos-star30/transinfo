# Release v2.0.0

发布日期：2026-04-02

这是当前项目的全新主版本发布，标记为 v2.0.0。

## 本次版本包含

- 路径搜索、字段映射、文本、逻辑链路统一到数据库技术名大写字段。
- 首页 Source 显示改为 `SOURCE + SOURCESYS`，避免 `SOURCENAME` 误导。
- 修复候选路径数量回退、映射不显示、诊断状态错误等关键回归。
- 新增全局 `Field info` 开关，显示 `Data Type / Length / Decimals`。
- 优化导出模板，只保留 `Aligned Mapping`，并支持模板缺失时自动回退。
- 新增 `RSTRANT` 导入卡片，支持 `TR Name` 在步骤区与导出中展示。
- 增加本地 / 线上环境隔离，降低部署后再启动本地时串接口、串数据库的风险。
- 补齐本地启动脚本、部署清单、路径逻辑与表关系说明。

## 当前发布口径

- 前端：Netlify
- 后端：Render Web Service（Docker）
- 数据库：TiDB Cloud 或其他 MySQL 兼容托管库

## 版本落点

- 前端运行时版本：`2.0.0`
- 页面版本角标：`v2.0.0`
- 静态资源缓存戳：`20260402-v2.0.0`

## 上线后建议核对

1. 登录、`/auth/me`、路径搜索是否返回 200。
2. 选定路径后，映射 / 文本 / 逻辑是否正常显示。
3. `RSTRANT` 导入后，`TR Name` 是否出现在 Step 标题下方。
4. Excel 导出是否包含最新字段信息与转换文本。
